What is a callback function and why will you use it?

A function is said to be a callback function. when the function passed into another function as an argument.
We need the callback function because many JavaScript actions are asynchronous, which means the function doesn't stop running until they're done, as you're probably used to. Instead, it will execute in the background while the rest of the code runs.
